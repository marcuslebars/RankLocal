# RankLocal

A web application for RankLocal - Local Growth Agency offering SEO, Ads, Web Design, and Social Media services.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```

## Project Structure

- `src/` - Source code
- `public/` - Static assets
- `tests/` - Playwright tests
